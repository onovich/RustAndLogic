## 当前位置插画占位图

- 资产文件：`D:\LabProjects\RustAndLogic\imgs\sample2.avif`
- 用途：
  - 作为界面左上角“当前位置”插画的占位图。
  - 它代表地点氛围图，不是任务列表配图，也不是普通侧栏装饰图。
- 布局定位：
  - 应放在左上角当前位置信息区域，作为地点标题、地点描述与任务区域上方的主视觉。
  - 它服务于“当前地点”识别，不应挪作资源图标、模块插图或调试面板背景。
- 图片使用方式：
  - 图片定位：`center / cover`
  - 要求居中显示，并允许裁剪铺满容器。
- 遮罩要求：
  - 使用渐变遮罩：
    `linear-gradient(to bottom, rgba(13, 8, 5, 0.2), #0D0805)`
  - 方向：自上而下
  - 顶部：`rgba(13, 8, 5, 0.2)`，只做轻微压暗
  - 底部：`#0D0805`，完全融入面板底色
  - 目标效果：插画下缘无缝衔接到面板背景，而不是硬切或单独悬浮
- 后续替换约定：
  - 将来即使替换成正式场景图，也应继承这一用途和这一套遮罩/定位规则，除非设计稿明确推翻。

## 使用约定

- 这些图是当前阶段的视觉与交互细节参考，不视为实现稿源文件。
- 后续如果用户继续在线程里追加新版参考图，应继续保存到本目录，并在这里追加说明。
- 涉及布局、按钮职责、面板折叠状态、对话框位置、调试信息显隐时，以这些参考图为优先依据。
- 涉及编辑器诊断、候选词下拉、跳转行为、运行时错误中断提示时，以 `ui-wireframe-v3-*` 为优先依据。
- 涉及左上角当前位置插画时，以本节对 `sample2.avif` 的用途、位置和遮罩规则为优先依据。
