# UI 设计参考图

这些图片来自当前 Codex 线程里用户直接提供的参考图，作为后续交互文档与界面细化的基准。

## 文件清单

- `ui-wireframe-v1.png`
  - 首版整体 UI 稿。
  - 重点是四栏结构、地图主舞台、代码区与右侧系统栏的关系。

- `ui-wireframe-v2-1.png`
  - 第二轮 UI 稿。
  - 补充了演出区底部对话框、右下角调试面板折叠态，以及语言切换入口。

- `ui-wireframe-v2-2.png`
  - 第二轮 UI 稿的设计讨论截图。
  - 记录了按钮语义、调试面板展开/折叠状态、语言切换按钮文案与位置等补充说明。

## 使用约定

- 这些图是当前阶段的视觉与交互细节参考，不视为实现稿源文件。
- 后续如果用户继续在线程里追加新版参考图，应继续保存到本目录，并在这里追加说明。
- 涉及布局、按钮职责、面板折叠状态、对话框位置、调试信息显隐时，以这些参考图为优先依据。
