# UI 设计参考图

这些图片来自当前 Codex 线程里用户直接提供的参考图，作为后续交互文档与界面细化的基准。

## 文件清单

- `ui-wireframe-v1.png`
  - 首版整体 UI 稿。
  - 重点是四栏结构、地图主舞台、代码区与右侧系统栏的关系。

- `ui-wireframe-v2-1.png`
  - 第二轮 UI 稿。
  - 补充了演出区底部对话框、右下角调试面板折叠态，以及语言切换入口。

- `ui-wireframe-v2-2.png`
  - 第二轮 UI 稿的设计讨论截图。
  - 记录了按钮语义、调试面板展开/折叠状态、语言切换按钮文案与位置等补充说明。

- `ui-wireframe-v3-editor-interactions.png`
  - 编辑器交互补充稿。
  - 明确了代码区底部诊断面板样式，以及报错/预警列表的呈现方式。
  - 补充了候选词下拉、接受/关闭提示、点击跳转等更细的编辑器交互方向。

- `ui-wireframe-v3-runtime-halt.png`
  - 运行时中断提示稿。
  - 用于循环执行过程中发生物理阻挡、逻辑错误或资源不足时，在地图左下角显示中断原因与建议处理文案。

## 使用约定

- 这些图是当前阶段的视觉与交互细节参考，不视为实现稿源文件。
- 后续如果用户继续在线程里追加新版参考图，应继续保存到本目录，并在这里追加说明。
- 涉及布局、按钮职责、面板折叠状态、对话框位置、调试信息显隐时，以这些参考图为优先依据。
- 涉及编辑器诊断、候选词下拉、跳转行为、运行时错误中断提示时，以 `ui-wireframe-v3-*` 为优先依据。
