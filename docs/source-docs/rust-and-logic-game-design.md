《Rust & Logic》游戏设计文档
1. 游戏标识

英文名称： Rust & Logic

中文备选：

《锈迹逻辑》

《废码黎明》

《拾荒法则》

《齿轮与伪码》

核心标签： 自动化放置 (Idle) | 编程解谜 (Programming) | 异步对抗 (Arena) | 复古科幻 (Retro-SciFi)

2. 核心玩法循环 (Core Loop)

游戏的整体节奏由“低频长线”的拾荒与“高频短线”的战斗交织而成。

编写与部署 (Build & Deploy): 玩家在主基地为拾荒机器人编写 TapeScript。配置完毕后，将其投入废土扇区。

挂机拾荒 (Idle Scavenge): 机器人在废土上按照代码自动游荡、收集“废铁”和“电芯”。此阶段支持基于底层极速推演的真实离线收益计算。

升级与拓展 (Upgrade): 利用资源扩展机器人的逻辑内存（解锁代码行数）、解锁新指令固件、升级硬件与装甲。

角斗与拓荒 (Arena & Explore): 组装重型战斗机甲，携带战术代码挑战地下城 Boss 或与其他玩家的机甲“幽灵”进行异步角斗，解锁新地图。

3. 核心资源系统

废铁 (Scrap)： 基础材料，升级机器人的机械部件（速度、负重、装甲）。

电芯 (Cells)： 能源单位，激活高级雷达和武器模块。

记忆晶片 (Memory Shards)： 极其稀有的升级资源，直接决定机器人可执行的指令行数上限（即逻辑内存扩容），构成最核心的数值追求。

4. 视听美学 (Aesthetics)

视觉风格： 废土侘寂 (Wasteland Wabi-sabi) 结合 CRT 显示器复古风。主色调为锈红色、废土黄、辅以低饱和度的荧光绿。

UI设计： 游戏采用拟真控制台界面，编程体验像是在维护一台受损的机器人逻辑核心；指令以冷静、克制的系统行写入内存。

听觉体验： 机械感 ASMR。代码执行时伴随清脆的继电器开合声与机械齿轮转动声。

5. 游戏终局 (End-Game)

多机协同流水线： 后期解锁无线电发射器模块，玩家需编写复杂的信号传递代码（Broadcast / Wait），让多台机器人在危险区协同搬运。

电子战天梯角斗： 上传自己的机甲配置，在服务器天梯上与其他玩家进行纯粹的逻辑较量。敌方机甲可能发射逻辑病毒（随机重置 PC 指针），考验玩家代码的“容错冗余度”。
